# Print Frame Generator

印刷用アートボード（フレーム）を生成する Figma プラグインです。用紙サイズ・向き・解像度・塗り足しを指定すると、トンボ相当のカットライン／セーフラインのガイド付きフレームを自動生成します。

## コンセプト

Figma で作ったデザインを、そのまま Illustrator の印刷データに貼り込める「印刷可能な画像セット」として書き出すためのフレームを用意することが目的です。

Figma 標準の `@2x` といった倍率指定の書き出しでは、配置した画像が実寸で低解像度でのノイズ（解像度不足）を起こしていないかを把握しづらいという問題があります。また、塗り足し（bleed）を正確にコントロールできる手段も乏しいのが実情です。

このプラグインは、**実際の用紙サイズ・解像度 (dpi) を起点にピクセル寸法を決め、塗り足しとセーフラインをガイドとして明示する**ことで、印刷入稿に耐えるデータを Figma 上で正確に組み立てられるようにします。

## 主な機能

- **用紙サイズ選択**: A3 / A4 / A5 / A6 / 名刺（日本・欧米） / 任意サイズ（幅・高さを mm で入力）
- **向き**: 縦 (portrait) / 横 (landscape)
- **解像度**: 350 / 300 / 250 / 200 / 150 dpi
- **塗り足し (bleed)**: プリセット (3mm / 5mm) または任意の数値を入力（0〜30mm）
- **セーフライン**: 任意指定。デフォルトは塗り足しと同じ値に同期
- **仕上がりサイズのプレビュー**: 設定に応じた出力ピクセル数をリアルタイム表示

生成されるフレームの内容:

- 白背景のフレーム（仕上がりサイズ + 塗り足し両側分）
- **カットライン**（マゼンタ 30% / 塗り足し位置）
- **セーフライン**（シアン 30% / 塗り足し + セーフライン位置）
- ガイド類は `guide` グループにまとめられ、ロック状態で生成されます

## 用紙サイズ定義

| サイズ | 寸法 (mm) |
| --- | --- |
| A3 | 297 × 420 |
| A4 | 210 × 297 |
| A5 | 148 × 210 |
| A6 | 105 × 148 |
| 名刺 日本 | 55 × 91 |
| 名刺 欧米 | 51 × 89 |

## ディレクトリ構成

```
figma-print-frame/
├── manifest.json      # Figma プラグインのマニフェスト
├── package.json       # ビルドスクリプト・依存関係
├── tsconfig.json      # TypeScript 設定
├── src/
│   ├── code.ts        # プラグイン本体（フレーム生成ロジック）
│   └── ui.html        # プラグインの UI
└── dist/
    └── code.js        # ビルド成果物（manifest の main が参照）
```

## セットアップ

```bash
git clone git@github.com:t4kuya6e/figma-print-frame.git
cd figma-print-frame
npm install
```

## ビルド

```bash
# 一度だけビルド
npm run build

# ソース変更を監視して自動ビルド
npm run watch
```

`src/code.ts` を esbuild でバンドルし、`dist/code.js` を出力します（manifest の `main` が参照）。UI は `src/ui.html` を直接読み込むため、ビルド対象外です。

## Figma への読み込み

1. `npm run build` でビルドする
2. Figma デスクトップアプリを開く
3. メニューから **Plugins → Development → Import plugin from manifest...** を選択
4. このフォルダの `manifest.json` を指定する
5. **Plugins → Development → Print Frame Generator** で起動

## 使い方

1. プラグインを起動する
2. 用紙サイズ・向き・解像度・塗り足し・セーフラインを設定する
3. **生成** ボタンを押す
4. 現在のページにガイド付きフレームが生成され、ビューポートが移動・選択された状態になる

生成されるフレーム名は `{用紙サイズ}_{向き}_{dpi}dpi_bleed{塗り足し}mm` の形式です（例: `A4_縦_350dpi_bleed3mm`）。

## 技術仕様

- mm → px 変換: `Math.round(mm * dpi / 25.4)`
- API: Figma Plugin API 1.0.0
- 対象エディタ: `figma`
- ビルドターゲット: ES2017

## 開発メモ

- `manifest.json` の `main` は `dist/code.js` を指すため、`src/code.ts` を編集したら必ずビルドが必要です。
- UI とプラグイン本体は `postMessage` (`pluginMessage`) でやり取りします。
